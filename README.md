# PRNN
Codes written for Pattern recognition and neural networks course, spring semester 2026.

1.EM-SGD: Codes for Expectation Maximization, Stochastic GD, Gaussian Mixture Models, SVM.

2.CNN-RNN-Transformer-LSTMs: MLPs, CNNs, RNNs, LSTMs and GRUs, Transformers and attention.

3. Tree-generative-Policygrad: Random Forests, VAE, GANs, Sim-CLR, and Policy gradient.
